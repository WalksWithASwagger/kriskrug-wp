## Deploy order

### Step 1 — Code Snippet (query param canonicalization)

1. wp-admin → Snippets → Add New
2. Title: `GSC404 query param canonicalize`
3. Paste body from [`fixes/gsc-404-query-param-canonicalize.php`](../fixes/gsc-404-query-param-canonicalize.php) **without** the opening `<?php` tag
4. Run everywhere, priority default, **Active**
5. Keep repo copy in sync if edited in wp-admin

### Step 2 — Redirection import

1. wp-admin → Tools → Redirection → Import/Export
2. Import [`fixes/gsc-404-redirection-import-2026-06.json`](../fixes/gsc-404-redirection-import-2026-06.json)
3. Confirm 18 new rules titled `GSC404 ...` appear and are enabled
4. Do **not** disable existing `/work/` or `/services/` rules

### Step 3 — Post-deploy curl matrix

Run after deploy (expect 301 to targets):

```bash
# Share param → canonical post (was 302 to x.com)
curl -sI "https://kriskrug.co/2006/08/28/barcampearth-a-local-report/?share=twitter&nb=1" | grep -E 'HTTP|location'

# Deleted category → blog
curl -sI "https://kriskrug.co/category/art/" | grep -E 'HTTP|location'
curl -sI "https://kriskrug.co/category/art/?amp=1" | grep -E 'HTTP|location'

# Drupal subpath → parent
curl -sI "https://kriskrug.co/2007/08/07/bryght-world-tour-2007/events/barcamp-vancouver-2007" | grep -E 'HTTP|location'

# Dead IDs → blog
curl -sI "https://kriskrug.co/?p=87" | grep -E 'HTTP|location'

# Mapped dead pages
curl -sI "https://kriskrug.co/quote-request/" | grep -E 'HTTP|location'
curl -sI "https://kriskrug.co/2023/09/28/future-in-review-podcast/" | grep -E 'HTTP|location'
curl -sI "https://kriskrug.co/shrine" | grep -E 'HTTP|location'

# Pagination overflow
curl -sI "https://kriskrug.co/category/field-notes/page/3/" | grep -E 'HTTP|location'
```

### Step 4 — GSC Validate fix (manual follow-up)

1. Google Search Console → **Pages** → **Not found (404)**
2. Click **Validate fix**
3. Do **not** use bulk Removals for URLs you've fixed with 301s
4. Optional: Removals only for junk probes if they persist after 30 days
5. Re-check affected count in **2–4 weeks** (GSC lags 24–72h+)
