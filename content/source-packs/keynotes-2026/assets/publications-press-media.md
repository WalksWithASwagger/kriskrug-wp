# Publications press media

Status: review assets for the Publications page overhaul. These files are local
source copies for layout review and a later WordPress Media Library upload.

## Publication gate

- Do not deploy `wp-payloads/publications.html` with third-party image URLs.
- Upload the approved files to the KrisKrug.co WordPress Media Library first.
- Preserve the source credit in the attachment caption and alt text.
- Replace each third-party `src` with its WordPress media URL.
- Re-run the desktop, mobile, link, and cache-bypassed checks after replacement.
