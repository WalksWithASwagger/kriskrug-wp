## Editorial Rules

- Use public videos, published owned-site material, and verified local source-pack artifacts only.
- Do not publish transcript dumps. Use transcripts for structure, paraphrased themes, and short reviewed excerpts only.
- Avoid naming private clients or private sessions unless there is public evidence.
- Use YouTube thumbnails and local thumbnail files as planning references. For live WordPress, upload/select WP-hosted media or embeds intentionally.
- Before a live draft moves out of this pack, run the privacy scan from `../video-research/README.md` and review names/client claims in context.
