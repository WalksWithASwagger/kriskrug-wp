## Final Select Intake

When final Midjourney selects arrive, add them to the future microsite repo under:

`public/assets/final-selects/`

Recommended naming:

- `wcfc-hero-01.webp`
- `wcfc-chapter-01-01.webp`
- `wcfc-chapter-02-01.webp`
- `wcfc-process-01.webp`

Each final image needs:

- Asset ID
- Filename
- Prompt ID from `prompt-pack.md`
- Alt text
- Caption
- Credit/provenance
- Approval status
- Intended chapter
- Preferred crop notes
