# World Cup Fashion Cake 2026 Source Pack

Implementation-ready handoff pack for the Becker x KK Vancouver World Cup 2026 AI fashion microsite and companion kriskrug.co blog draft.

This is a Track A content/source pack. It does not create or update any live WordPress content, and it does not implement the standalone microsite. It gives the next agents enough context to build the site, ingest final Midjourney selects when they arrive, and prepare a publishable blog reflection without guessing at the collaboration story.

## Creative Brief

Kris/KRUG and Tania Becker used to collaborate in the fashion industry. Kris was a photographer. Tania is a high-end LA-based avant-garde fashion hairstylist with a West Coast fashion eye. This project turns their chat about mood boards, prompt calibration, Midjourney, Lightroom selects, and motion tests into two artifacts:

- A standalone, vibe-coded, futuristic photo essay microsite about Vancouver World Cup 2026 fashion energy.
- A kriskrug.co blog post about the meta-story: old fashion collaborators using an army of AI tools to make strange new fashion media together.

The microsite should feel like Lost Angels meets Vancouver: LA gloss, Vancouver rain, stadium lights, Pacific fog, sculptural hair, soccer-coded couture, and robot-assisted creative process.
