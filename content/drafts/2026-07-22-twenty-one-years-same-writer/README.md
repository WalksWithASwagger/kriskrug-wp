# Twenty-One Years, and I Write Exactly the Same

Track A long-form article packet prepared from the private Claude voice-forensics artifact and the committed kriskrug.com archive.

## Files

- `post.md`: canonical 2,000-word article draft and WordPress frontmatter.
- `post.html`: bespoke dark forensic-report layout, packaged as one Gutenberg Custom HTML block with scoped CSS.
- `artifact-analysis.md`: editorial, evidence, and design assessment of the supplied artifact.
- `source-manifest.md`: primary sources, claim status, privacy boundary, and numeric drift ledger.
- `seo-meta.md`: title, description, keywords, excerpt, and schema recommendation.
- `internal-links.md`: proposed links and public-route checks.
- `image-brief.md`: featured-image direction and generation prompt.
- `social-content.md`: draft-only LinkedIn, X, and Bluesky copy.
- `publish-gate.md`: private-draft and publication requirements.
